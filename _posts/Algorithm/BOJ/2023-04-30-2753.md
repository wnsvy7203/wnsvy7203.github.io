# **문제**
[acmicpc.net/problem/2753](https://www.acmicpc.net/problem/2753)
![Imgur](https://i.imgur.com/xv9ZcJb.png)

*윤년의 개념을 알고 분기처리만 신경 쓰면 풀 수 있는 문제다.*

- 기본적으로 4의 배수면 윤년이고,
  - 그 중에서 100의 배수인 해는 평년
  - 다시 400의 배수인 해는 윤년

# **정답 코드**
```c++
#include <iostream>

using namespace std;

int n;

int main()
{
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    cin >> n;

    if (!(n % 4))
        if (!(n % 400))
            cout << 1;
        else if (n % 100)
            cout << 1;
        else
            cout << 0;
    else
        cout << 0;
}
```
