# **문제**
[acmicpc.net/problem/2675](https://www.acmicpc.net/problem/2675)
![Imgur](https://i.imgur.com/RZcWr4o.png)

*주어진 횟수만큼, 해당 인덱스의 문자를 출력하면 되는 문제*

# **정답 코드**
```c++
#include <iostream>
#include <string>

using namespace std;

int T;

int main()
{
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    cin >> T;

    int R;
    string S;

    for (int t = 0; t < T; t++)
    {
        cin >> R >> S;

        for (int i = 0; i < S.length(); i++)
            for (int j = 0; j < R; j++)
                cout << S[i];
        cout << '\n';
    }
}
```
